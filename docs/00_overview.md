# 项目总览

## 项目定位

Become Engineer 是一个面向公众的学习平台与知识库，也是一套服务个人长期成长的工程学习系统。

项目不追求简单堆链接，而是把来自免费资源的内容进行筛选、验证、纠错、重组和实践化，最终形成可阅读、可练习、可复盘、可持续演进的公开学习资料。

## 核心目标

- 建立覆盖工程与 AI 的长期学习地图。
- 收集高质量免费资源，并记录来源、适用阶段、优缺点和使用建议。
- 将零散知识沉淀为结构化笔记、路线图、练习题、项目任务和复盘记录。
- 通过公开仓库形成可被他人学习、审阅和贡献的知识产品。
- 保留个人学习空间，但不把私人资料暴露到公开仓库。

## 内容边界

### 公开内容

- 学习路线与阶段目标
- 主题笔记与知识卡片
- 资源清单与评估
- 可公开的代码示例
- 项目练习与实验记录
- 错误修订与版本说明

### 私有内容

- 个人原始学习笔记
- 未整理草稿
- 账号、密钥、课程登录信息
- 下载的受版权保护资料
- 不适合公开的个人复盘

私有内容统一放入本地 `个人学习/` 目录，并由 `.gitignore` 排除。

## 首批学习领域

- Python
- C++
- 前端开发
- 后端开发
- 数据库
- 机器学习
- 深度学习
- 强化学习
- LLM
- Agent

## 内容生产原则

- 先验证，再沉淀。
- 先结构化，再扩写。
- 保留来源，不搬运侵权内容。
- 公开内容面向他人阅读，个人草稿保留在私有区。
- 每个主题尽量包含概念、路线、资源、练习和项目。
