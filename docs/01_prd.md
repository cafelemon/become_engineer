# 产品需求文档

## 背景

工程与 AI 学习资源很多，但常见问题是路径混乱、质量参差、内容重复、实践不足、缺少复盘。这个项目希望把免费资源整理为一套可长期使用的公开学习平台。

## 用户

### 主要用户

- 项目作者本人：用于系统学习、复盘、查漏补缺和项目实践。
- 自学工程与 AI 的学习者：需要清晰路线、可靠资源和可执行任务。

### 次要用户

- 想快速了解某一技术栈的人。
- 想贡献优质免费资源或修订错误的人。

## 用户需求

- 我想知道从零到可实践某个方向应该怎么学。
- 我想找到经过筛选的免费学习资源。
- 我想知道每个资源适合什么阶段、有什么坑、如何使用。
- 我想通过练习和项目验证自己是否真的掌握。
- 我想看到学习内容的来源、版本和修订记录。

## 核心功能

- 学习地图：按方向、阶段和前置知识组织内容。
- 资源库：收集免费资源并做质量评估。
- 笔记库：沉淀公开可读的知识笔记。
- 练习库：提供题目、实验和项目任务。
- 纠错机制：记录错误、争议、修订来源和结论。
- 进度记录：记录项目建设过程与下一步任务。

## 非目标

- 不直接搬运付费课程或受版权保护内容。
- 不做只堆链接的导航站。
- 不在公开仓库保存个人隐私学习资料。
- 初期不追求复杂账号系统、社交系统或在线判题系统。

## 成功标准

- 仓库结构清晰，新读者能在 5 分钟内理解项目目标与入口。
- 每个公开主题都有来源、学习目标、核心笔记和练习建议。
- 私有目录不会被 git 跟踪。
- 资源引用符合版权和开源边界。
- 项目可以持续按版本迭代。
