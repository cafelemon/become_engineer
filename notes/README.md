# 公开笔记

这里存放整理后的公开学习笔记。

个人草稿请先放在本地 `个人学习/`，经过筛选、重写、补充来源和隐私检查后，再进入本目录。

## 笔记要求

- 能映射到至少一条学习主线。
- 标明在统一路线中的位置、前置知识和下一步；新学习者不从笔记索引随机起步。
- 优先解释概念、误区、实践方法和自测问题。
- 不直接搬运课程、书籍或文章原文。
- 对不确定内容标记为待验证。
