# 公开笔记

这里存放整理后的公开学习笔记。

个人草稿请先放在本地 `个人学习/`，经过筛选、重写、补充来源和隐私检查后，再进入本目录。

## 笔记要求

- 能映射到至少一条学习主线。
- 标明在统一路线中的位置、前置知识和下一步；新学习者不从笔记索引随机起步。
- 优先解释概念、误区、实践方法和自测问题。
- 按内容类型选择 Mermaid、表格、代码、输出、正反例或实验，不为装饰机械加图。
- 可复制示例写明环境、文件、依赖、命令、预期输出和失败路径。
- 使用AI时记录任务约束、产出摘要、人工修改和验证证据；核心内容设置手动检查点。
- 明确笔记关联的是微练习、阶段作品还是长期项目里程碑，纯理论笔记说明能力去向。
- 不直接搬运课程、书籍或文章原文。
- 对不确定内容标记为待验证。

新增或扩写笔记遵循[课程内容规范](../docs/08_content_standard.md)和[笔记模板](https://github.com/cafelemon/become_engineer/blob/main/templates/note_template.md)。
