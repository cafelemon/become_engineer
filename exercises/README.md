# 练习库

这里存放题目、实验、代码练习和阶段性自测。

练习应尽量说明：

- 目标
- 输入
- 输出
- 约束
- 验收标准
- 提示
- 延伸任务

## 主线映射

每个练习应标注所属主线和适合阶段，方便从学习路线直接跳转到练习。

新学习者不要从本页任选练习，应先按[面向小白的统一学习路线](../learning-paths/beginner-roadmap.md)完成对应前置阶段。
