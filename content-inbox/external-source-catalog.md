# 外部来源目录

这里登记“当前不下载，但已经确认结构和用途”的外部学习资料。

来源目录不是正式课程，也不是待全文搬运清单。建设某条路线、笔记或项目里程碑时，先从这里定位候选章节，再按需读取、核查和二次加工。

## 使用规则

1. 先从我们的六条主线和项目里程碑提出具体内容需求。
2. 在来源目录中选择最小必要章节，不整站下载。
3. 记录来源 URL、访问日期、版本或提交、许可证和选取原因。
4. 原始内容如需本地处理，放入 `source-materials/raw/` ignored 目录。
5. 对照当前官方文档、课程官网和可运行实验核查。
6. 最终只将独立重写的路线、笔记、练习、项目步骤或资源评测放入公开目录。

### 对本项目的主要价值

- 为工程、C++、通用算法、系统、网络、数据库和 Web 各阶段提供已登记的课程候选，具体内容按统一路线逐步加工。
- 提供课程难度、作业和项目体验建议，帮助我们筛选而不是盲目收集。
- 帮助核查课程内容和难度，但不改变[面向小白的统一学习路线](../learning-paths/beginner-roadmap.md)。

### 按需取用优先级

1. 建设工程基础时：使用指南、Missing Semester、Git、GitHub、Make/CMake、Docker、工作流和信息检索。
2. 建设 C++ 时：Stanford CS106L、CS106B/X 等课程评测页。
3. 建设 CS 核心时：CS61B、CSAPP、操作系统、网络、数据库和软件工程页面，算法只取通用算法与复杂度相关内容。
4. 建设 Web 时：Web 开发栏目中的课程比较页，并优先寻找PostgreSQL/psql实践材料，再补MySQL对照。
5. 建设数学前置时：线性代数、概率与离散数学栏目。

### 不做的事情

- 不复制其完整站点结构作为我们的路线。
- 不批量下载它链接的课程视频、教材、作业和学生实现。
- 不因为某门课程被推荐，就跳过课程官网、版本、作业许可和可访问性核查。

### 基本信息

- 当前入口：https://fastapi.tiangolo.com/zh/python-types/
- 源码：https://github.com/fastapi/fastapi
- 类型：官方框架文档、教程、示例和 API 参考。
- 语言：英文为源文档，中文翻译按章节提供。
- 访问状态：公开可访问，无需登录。
- 许可：MIT。
- 当前状态：`cataloged, official-source, fetch-on-demand`

### 内容地图

| 学习阶段 | 主要内容 | 映射到本项目 |
| --- | --- | --- |
| Python 前置 | 类型提示、异步、环境变量、虚拟环境 | 编程语言、工程基础 |
| API 入门 | 路径参数、查询参数、请求体、Pydantic 模型 | Web 全栈 |
| 数据契约 | 字段校验、嵌套模型、响应模型、状态码 | Web 全栈、项目 2 服务化 |
| 文件与错误 | 表单、文件、错误处理、编码与局部更新 | Web 全栈 |

| 测试与调试 | TestClient、异步测试、依赖替换、调试 | 工程基础、所有 API 项目 |
| 部署 | HTTPS、进程、Worker、Docker、云部署 | 项目交付阶段 |
| OpenAPI | Schema、回调、Webhook、客户端生成 | API 契约与集成 |

### 对本项目的主要价值

- Python 类型提示页可用于 Python 工程化课程中的类型章节。
- FastAPI 教程可服务结构化数据 ML 项目的推理 API 阶段。
- 依赖、安全、测试、SQL 和大型应用章节可服务智能学习助手的 Web/API 工程，但数据库实践仍以PostgreSQL/psql优先。
- 流式响应、SSE 和 WebSocket 可在 LLM 流式交互确有需要时使用。

### 按需取用顺序

1. Python 内容分析工具涉及类型设计时：只取类型提示和环境相关章节。
2. 结构化数据 ML 进入 P2.7 时：取请求体、响应模型、错误处理、测试和应用结构。
3. 智能学习助手进入 API 阶段时：取依赖注入、SQL数据库、安全、测试、流式输出和部署，并对照PostgreSQL实践。
4. 只有实现确有需要时，才取高级 OpenAPI、Webhook、WebSocket 或代理部署内容。

### 取用注意事项

- 优先读取当前官方英文源文档，并确认中文页面是否同步。
- 文档示例引用 `docs_src/` 中经过测试的代码；按需取页时同时定位对应最小示例。
- 安装命令、Pydantic 版本、CLI 和部署建议可能变化，每次加工都重新核查。
- FastAPI 是 Web 后端实现工具，不能替代 HTTP、数据库、认证、测试和系统设计基础课。
