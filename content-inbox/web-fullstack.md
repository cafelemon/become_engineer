# Web 全栈素材队列

## 处理重点

- JavaScript/TypeScript、前端、后端、PostgreSQL/psql、MySQL对照、API、部署和产品化交付。
- 后端初期优先Python，Java后端作为补充方向。
- 优先筛选可以转化为完整应用项目的材料。
- SQL作为共同语言学习，但实践资料优先围绕PostgreSQL和`psql`组织；MySQL按真实场景补充差异。
