## 路线写作原则

- 先写学习目标，再列资源。
- 明确该主线在统一路线中的进入阶段和解锁条件。
- 每个阶段都要有可验证产出。
- 资源只收录免费、可访问、可追溯的内容。
- 不确定内容标记为待验证。
- 避免只堆链接，必须说明为什么学、怎么学、学到什么程度。
