# AI 基础

## 定位

AI 基础线用于建立机器学习、深度学习和强化学习的核心理解。目标是能读懂基本模型、完成实验、评估结果，并知道模型适合解决什么问题。

## 适合人群

- 已具备 Python 基础，准备进入 AI 学习的人。
- 想从应用走向模型理解的人。
- 想为 LLM 和 Agent 学习补齐基础的人。

## 阶段划分

1. 数学与数据基础：线性代数、概率统计、数据处理、评估指标。
2. 机器学习：监督学习、无监督学习、特征工程、模型评估。
3. 深度学习：神经网络、反向传播、CNN、RNN、Transformer。
4. 强化学习：MDP、价值函数、策略、探索与利用。
5. 实验工程：Notebook、训练记录、复现实验、结果分析。

## 推荐学习顺序

1. 先用 Python 和数据处理完成小实验。
2. 用机器学习建立数据、特征、模型、评估闭环。
3. 用深度学习理解神经网络训练过程。
4. 用强化学习作为扩展方向，不急于和主线混在一起。
