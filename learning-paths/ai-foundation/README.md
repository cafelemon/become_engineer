# AI 基础

## 定位

AI 基础用于建立从数据、模型、训练到评估的完整能力，覆盖机器学习、深度学习和NLP/Transformer共同基础。目标不是运行现成Notebook，而是能解释假设、复现实验、比较模型并分析失败。

这里的“算法”指模型算法。数据结构、复杂度、排序、树和图等通用算法属于[CS 核心](../cs-core/README.md)。

## 在默认路线中的位置

- 进入条件：Python工程化、CS必修主干和Web必修主干完成。
- 必修主干完成后：解锁LLM/Agent必修主干，并能进入AI推荐深化。
- 推荐深化和专业选修：不阻塞LLM/Agent，但部分专业方向互相依赖。
- 完整关系见[完整课程地图](../curriculum-map.md)。

## 适合人群

- 想从零建立机器学习和深度学习能力的人。
- 想理解NLP、Transformer和LLM前置原理的人。
- 想继续进入强化学习、视觉、多模态或生成模型的人。
- 想把模型实验推进到可复现工程的人。

## 必修主干

| 模块 | 知识范围 | 实践与验收 |
| --- | --- | --- |
| 数学与数据 | 线性代数、概率统计、微积分、梯度、数据清洗和基本可视化 | 能计算和解释模型使用的核心量，并完成数据质量检查 |
| 实验与评估 | 训练/验证/测试划分、指标、基线、交叉验证、数据泄漏和误差分析 | 建立固定实验协议并证明结果可重复 |
| 机器学习 | 监督学习、无监督学习、特征工程、线性模型、树模型和模型比较 | 在结构化数据上完成从基线到推理的闭环 |
| 深度学习 | 张量、神经网络、反向传播、优化器、正则化、训练与验证 | 实现并训练基础网络，记录曲线、指标和失败分析 |
| NLP与序列模型 | 文本表示、传统文本模型、RNN/CNN、Attention和Transformer | 在同一文本任务上比较传统模型、深度模型和Transformer |
| 实验工程 | 环境、随机种子、配置、数据版本、模型保存、日志和复现说明 | 他人能按照说明复现实验并得到可解释的误差范围 |

### 必修完成标准

- 能建立简单基线，并说明为什么选择该指标。
- 能识别数据泄漏、过拟合、类别不平衡和无效比较。
- 能完成传统机器学习、基础深度学习和Transformer的代表性实验。
- 能提交配置、数据说明、训练记录、指标、误差分析和推理结果。
- 能说明模型能力边界，而不是只报告一个最好分数。

## 推荐学习顺序

1. 先完成数学、数据和可靠评估。
2. 用机器学习建立完整实验闭环。
3. 再进入深度学习、NLP和Transformer。
4. 完成复现实验和误差分析后解锁LLM/Agent。
5. 根据目标选择推荐深化和专业选修，不要求一次铺开全部方向。
