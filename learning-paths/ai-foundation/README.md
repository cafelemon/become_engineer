# AI 基础

## 定位

AI 基础线用于建立机器学习、深度学习、NLP/Transformer和强化学习的核心理解。目标是能读懂基本模型、完成实验、评估结果，并知道模型适合解决什么问题。

本主线在Python工程化和CS最小核心完成后进入；数学与评估基础是本阶段最先完成的单元。

这里的“算法”指模型算法，不是CS阶段的通用算法。通用算法负责解释数据结构、复杂度和程序性能；模型算法负责解释数据、特征、训练、优化、泛化和决策。

## 适合人群

- 已具备 Python 基础，准备进入 AI 学习的人。
- 想从应用走向模型理解的人。
- 想为 LLM 和 Agent 学习补齐基础的人。

## 阶段划分

1. 数学与数据基础：线性代数、概率统计、数据处理、评估指标。
2. 机器学习：监督学习、无监督学习、特征工程、模型评估。
3. 深度学习：神经网络、反向传播、CNN、RNN、Attention和Transformer。
4. 强化学习：MDP、价值函数、策略、探索与利用。
5. 实验工程：Notebook、训练记录、复现实验、结果分析。

## 推荐学习顺序

1. 先完成数据处理、数学前置和可靠评估方法。
2. 用机器学习建立数据、特征、模型和评估闭环。
3. 再进入神经网络、反向传播和PyTorch。
4. 通过NLP/Transformer项目比较模型演进，并为LLM/Agent建立关键前置。
5. 强化学习作为独立并行分支，不是LLM/Agent前置。

## 进入与离开条件

- 进入条件：Python工程化、基本复杂度与实验方法。
- 模型项目解锁：先完成线性代数、概率统计、梯度、数据划分和评估指标。
- LLM/Agent解锁：能够复现传统ML与深度学习模型，并提交指标、误差分析和复现说明。

详细顺序见[小白统一学习路线](../beginner-roadmap.md)。
