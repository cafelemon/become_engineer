# LLM/Agent

## 定位

LLM/Agent 线用于学习大模型应用和智能体系统，包括 Prompt、RAG、工具调用、评估、微调、Agent 框架和真实项目交付。

## 适合人群

- 已有 Python 或 Web 基础，想做 LLM 应用的人。
- 想理解 Agent 系统如何规划、调用工具和处理上下文的人。
- 想把 AI 能力接入真实业务或个人效率工具的人。

## 阶段划分

1. LLM 使用基础：模型能力、上下文、Prompt、结构化输出。
2. RAG：文档切分、向量检索、重排、引用、评估。
3. 工具调用：函数调用、外部 API、权限、错误恢复。
4. Agent：规划、记忆、状态、工作流、多 Agent 协作。
5. 工程化：评估集、日志、成本、延迟、安全、部署。

## 推荐学习顺序

1. 先做简单 Prompt 和结构化输出实验。
2. 再做一个小型 RAG 知识库。
3. 加入工具调用和可观测日志。
4. 最后进入 Agent 工作流和真实项目交付。
