# 变量、基本类型、输入输出

本节是 Python 起步的第一课。你会写出一个最小可运行脚本，并理解程序如何保存数据、显示结果和接收输入。

先不要追求写复杂程序。当前最重要的是建立一个闭环：写代码，运行代码，观察输出，记录结果。

## 前置知识

开始前应完成：

- [工程基础入门](../../engineering-foundation/stage-0/README.md)
- 能用编辑器创建 `.py` 文件。
- 能在终端进入文件所在目录。
- 能运行 `python3 --version` 或记录 Python 不可用时的错误。
- 能把代码和运行结果写进 Markdown 学习记录。

如果你还没有确认 Python 命令可用，请先完成工程基础入门中的开发环境和验证习惯内容。

## 学习目标

完成本节后，你应该能做到：

- 写出并运行一个最小 Python 脚本。
- 使用变量保存数据。
- 区分整数、浮点数、字符串和布尔值。
- 使用 `print()` 输出结果。
- 使用 `input()` 接收键盘输入。
- 理解 `input()` 得到的内容默认是字符串。
- 使用 `int()`、`float()`、`str()` 做最基本的类型转换。
- 根据常见错误信息判断问题位置。

## 学习顺序

按下面顺序学习：

1. 先运行第一行 Python。
2. 再用变量保存数据。
3. 接着认识几种基本类型。
4. 然后学习输出和输入。
5. 最后完成一个小脚本，把输入、转换和输出连起来。

## 第一行 Python

新建文件：

```text
variables_io.py
```

写入：

```python
print("Hello, Python")
```

在文件所在目录运行：

```bash
python3 variables_io.py
```

你应该看到：

```text
Hello, Python
```

这就是最小闭环：文件存在，代码能运行，输出符合预期。

## 变量是什么

变量可以理解为给一个值起名字。以后要用这个值时，直接使用名字。

```python
name = "Lemon"
age = 18

print(name)
print(age)
```

这里：

- `name` 是变量名。
- `"Lemon"` 是字符串值。
- `age` 是变量名。
- `18` 是整数值。

变量名应该能说明含义。比如 `age` 比 `a` 更清楚，`product_name` 比 `pn` 更适合学习阶段。

## 基本类型

先认识四种最常用的类型：

```python
count = 3
price = 19.9
title = "Python 起步"
is_finished = False

print(type(count))
print(type(price))
print(type(title))
print(type(is_finished))
```

常见含义：

| 类型 | 示例 | 用来表示 |
| --- | --- | --- |
| 整数 | `3` | 次数、年龄、数量 |
| 浮点数 | `19.9` | 价格、比例、带小数的数 |
| 字符串 | `"Python"` | 文本、名字、路径 |
| 布尔值 | `True`、`False` | 是或否、成立或不成立 |

Python 区分大小写。`True` 和 `False` 的首字母必须大写。

## 输出

`print()` 用来把结果显示出来。

```python
name = "Lemon"
age = 18

print("姓名：", name)
print("年龄：", age)
```

输出：

```text
姓名： Lemon
年龄： 18
```

学习阶段建议多使用 `print()`。它能帮你确认变量当前是什么值，程序走到了哪一步。

## 输入

`input()` 用来接收键盘输入。

```python
name = input("请输入你的名字：")
print("你好，", name)
```

运行后，程序会等待你输入内容。

注意：`input()` 得到的内容默认都是字符串。

```python
age_text = input("请输入你的年龄：")
print(type(age_text))
```

即使你输入 `18`，`age_text` 也是字符串，不是整数。

## 类型转换

如果要做数字计算，需要把字符串转成数字。

```python
age_text = input("请输入你的年龄：")
age = int(age_text)
next_year_age = age + 1

print("明年你", next_year_age, "岁")
```

常用转换：

| 写法 | 含义 |
| --- | --- |
| `int("18")` | 把可转换的字符串变成整数 |
| `float("19.9")` | 把可转换的字符串变成浮点数 |
| `str(18)` | 把数字变成字符串 |

如果输入内容不能转换，程序会报错。例如 `int("abc")` 会失败。当前先学会识别错误，后面会专门学习异常处理。

## 最小脚本

把下面代码保存为 `profile.py`：

```python
name = input("姓名：")
age_text = input("年龄：")
height_text = input("身高，单位米：")

age = int(age_text)
height = float(height_text)

print("学习档案")
print("姓名：", name)
print("年龄：", age)
print("明年年龄：", age + 1)
print("身高：", height)
print("是否已经成年：", age >= 18)
```

这个脚本已经包含：

- 输入。
- 变量。
- 字符串。
- 整数。
- 浮点数。
- 布尔判断结果。
- 输出。

## 常见错误与排查

| 错误 | 表现 | 怎么排查 |
| --- | --- | --- |
| 文件没有保存 | 运行后看不到最新输出 | 保存文件后重新运行 |
| 运行错目录 | 提示找不到文件 | 用 `pwd` 或 `Get-Location` 确认当前目录 |
| 变量名写错 | `NameError` | 检查变量名拼写和大小写 |
| 字符串没有引号 | `NameError` 或语法错误 | 文本值要用引号包住 |
| 字符串和整数直接相加 | `TypeError` | 先转换类型，或用 `print()` 分开输出 |
| 输入不能转成整数 | `ValueError` | 确认输入内容只包含可转换的数字 |
| `True`、`False` 小写 | `NameError` | 布尔值首字母必须大写 |
