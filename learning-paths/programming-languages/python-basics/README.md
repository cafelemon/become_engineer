# Python 起步

Python 起步的目标不是一次学完所有语法，而是建立完整的编程起点：会写小程序、会运行和拆分代码、会处理外部数据，也会根据错误与测试检查自己的判断。

完成工程基础入门后再进入这里。你需要继续使用前面建立的习惯：每次学习都记录目标、代码、运行结果、错误和判断依据。

## 前置要求

开始前应完成：

- [工程基础入门](../../engineering-foundation/stage-0/README.md)
- 能打开项目目录或自己的学习目录。
- 能使用终端运行简单命令。
- 能用编辑器创建和保存文本文件。
- 能写一份包含代码块和运行结果的 Markdown 学习记录。

如果你还不能稳定找到文件、运行命令或记录错误，请先回到工程基础入门补齐。Python 起步会开始写代码，前置工具不稳会让后面的问题变得很难判断。

## 学习目标

完成 Python 起步后，你应该能做到：

- 写出可以运行的 `.py` 文件。
- 使用变量保存数据。
- 理解数字、字符串、布尔值、列表、字典等常见类型。
- 使用条件和循环控制程序流程。
- 把重复逻辑整理成函数。
- 读取文本或 JSON 文件，并输出简单统计结果。
- 根据 traceback 定位到具体文件、行号和错误类型。
- 使用标准库测试覆盖正常、边界和失败场景。
- 为后续 Python 内容分析工具项目做好准备。

## 学习方式

每节课先让你看到程序能做什么，再用短例子解释其中的关系。你会在本机把例子跑起来，换成自己的数据，故意试一次常见错误，并把这次变化接到学习进度报告器上。

不要只复制代码。至少改一次名字、数据或处理方式，再看输出怎样变化。可以让 AI 帮你解释错误或给出候选写法，但最后仍要由你运行、判断和说明结果。

## 运行方式

本课程只使用 Python 标准能力，不需要第三方库。

你可以把练习代码放在自己的学习目录中，例如：

```text
python-learning/
├── variables_io.py
├── conditions_loops.py
├── study_report.py
├── study_records.py
├── study_records_from_json.py
├── main.py
├── analysis.py
├── data_io.py
├── reporting.py
├── tests/
│   └── test_*.py
└── data/
    └── study_records.json
```

常见运行方式：

```bash
python3 variables_io.py
```

Windows 上如果 `python3` 不可用，可以尝试：

```powershell
python variables_io.py
```

如果命令不可用，不要跳过。请把完整错误写进学习记录，再回到工程基础入门的开发环境和验证习惯内容排查。

## 完成标准

完成 Python 起步需要满足：

- 能独立创建并运行一个 `.py` 文件。
- 能解释变量名和值的关系。
- 能区分字符串、整数、浮点数和布尔值。
- 能使用 `if`、`for`、`while` 写出基本分支和循环。
- 能把重复代码提取成函数。
- 能选择列表或字典保存多条数据。
- 能读取一个文本或 JSON 文件，并输出统计结果。
- 能根据 traceback 找到错误发生的文件和行。
- 能为可预期输入错误给出清楚提示和非零退出码。
- 能使用 `unittest` 和临时目录验证正常、边界和失败行为。
- 能证明程序不修改输入文件，导入模块不启动主流程。
- 能写出一份包含代码、输入、输出和判断的学习记录。

满足以上标准后，完成并验收[学习进度报告器](../../../exercises/python-basics/study-progress-reporter/README.md)。这个阶段作品从第一课的学习档案 v0.1 开始，连续吸收七节课程，但不冒充大型商业项目。

## 完成后的方向选择

完成 Python 起步后，可以并行进入 [Python 内容分析工具](../../../projects/python-content-analysis/README.md) 的第一个里程碑。

这个项目会用 Python 处理 Markdown、JSON 和目录数据，但不会一次要求你完成全部工程化内容。CLI、日志、配置和 CI 会随着 Python 工程化课程逐步加码。

默认学习顺序先进入 [CS 起步](../../cs-core/README.md)，用已经会写的 Python 程序理解数据表示、位置、操作成本、文本和网格。完成共同基座和阶段作品后，再选择 Python 核心、C++ 起步、算法或其他方向。
