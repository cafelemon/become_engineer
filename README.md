# Become Engineer

Become Engineer 是一个公开学习平台项目，目标是把免费、可信、可复用的工程与 AI 学习资源整理成一套可长期维护的学习体系。

这个仓库面向公众开放，但也服务于我自己的学习。公开部分沉淀课程路径、知识笔记、实践项目、资源评估、错漏修订和阶段性成果；个人学习资料放在本地 `个人学习/` 目录中，并已加入 `.gitignore`，不会进入公开仓库。

## 内容范围

- 编程基础：Python、C++、工程化习惯、调试与测试
- Web 开发：前端、后端、API、部署与性能
- 数据基础：数据库、SQL、数据建模、数据工程
- AI 基础：机器学习、深度学习、强化学习
- LLM 与 Agent：大模型应用、提示词、RAG、工具调用、Agent 系统
- 学习方法：资源筛选、错题复盘、项目制学习、公开输出
