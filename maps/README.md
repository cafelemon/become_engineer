# 学习地图

这里存放 Become Engineer 的全局学习地图，用来回答三个问题：

- 先学什么，再学什么。
- 每条主线之间是什么关系。
- 学到什么程度可以进入实践项目。

## 入口

- [小白统一学习路线](../learning-paths/beginner-roadmap.md)：唯一默认学习顺序和阶段验收。
- [Become Engineer 总地图](become-engineer-map.md)
- [Python、AI 与 LLM 能力依赖图](python-ai-llm-dependency-map.md)
- [Python、AI 与 LLM 素材覆盖说明](python-ai-llm-content-coverage.md)

## 使用方式

- 新读者先看小白统一学习路线，再按阶段进入主线。
- 维护者新增内容时，先确认内容属于哪条主线和哪个阶段。
- 个人学习成果公开化时，先映射到地图位置，再选择沉淀到笔记、项目、资源评测或公开输出。
