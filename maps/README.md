# 学习地图

这里存放 Become Engineer 的全局学习地图，用来回答三个问题：

- 先学什么，再学什么。
- 每条主线之间是什么关系。
- 学到什么程度可以进入实践项目。

## 入口

- [开始学习](../learning-paths/beginner-roadmap.md)：四类用户入口、共同基座和方向摘要。
- [完整课程地图](../learning-paths/curriculum-map.md)：唯一公开课程控制面，包含状态流程、阶段总表和 55 节明细。
- [Become Engineer 总地图](become-engineer-map.md)：架构摘要。
- [Python、AI 与 LLM 能力依赖图](python-ai-llm-dependency-map.md)
- [Python、AI 与 LLM 素材覆盖说明](python-ai-llm-content-coverage.md)

## 使用方式

- 新读者先看开始学习页；需要选择方向、判断前置或查看建设状态时进入完整课程地图。
- 维护者新增内容时，先确认所属模块、六级深度、角色、前置和状态，并同步课程登记。
- 个人学习成果公开化时，先映射到地图位置，再选择沉淀到笔记、项目、资源评测或公开输出。
