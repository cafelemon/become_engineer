# Become Engineer 总学习地图

本页只保留架构摘要。课程层级、前置、状态、四类用户规则和当前 117 节课程明细，以[完整课程地图](../learning-paths/curriculum-map.md)为唯一权威版本。

## 服务对象

- 小白 · 兴趣学习。
- 有基础 · 兴趣学习。
- 小白 · 转行求职。
- 有基础 · 强化求职。

四类用户共用同一棵能力树：小白／有基础决定起点和补修，兴趣／求职决定是否叠加机考、项目证据和面试训练。

## 内容衔接摘要

```text
共同基座
  工程基础起步 → Python 编程起步 → CS 起步 → 首个可验证项目与方向选择

方向主干
  应用工程：Python 核心与工程化 → Web
  系统工程：C++ 起步 → C++ 核心 → 系统、并发、网络与性能
  算法：共同算法基础 → 算法核心 → 算法深化
  AI 模型：数学与数据 → 机器学习 → 深度学习 → 专业模型方向
  LLM 应用：Python 工程化 + Web/API → 模型使用 → RAG 与评估
  Agent 工程：Tool Calling → 有界工作流 → 状态、恢复、评估与安全
  设备系统：C 起步 + 系统能力 → MCU/RTOS、Linux/BSP、工业控制

跨线汇合
  设备平台 + AI 模型 → 边缘智能
  设备平台 + 强化学习 → 智能控制
  Agent 工程 + 模型／系统能力 → 微调、本地推理或多模态 Agent
```

C++ 不再作为所有学习者进入 CS 的强制前置，C 语言只进入设备系统方向。基础算法属于共同 CS 起步；完整算法核心、高阶算法和机考训练按兴趣或求职目标进入。LLM 应用与 AI 模型相邻但不同，应用 Agent 不强制完整机器学习和深度学习。
